# DIKWP RealityLoop Commons OS

Open mission deployment, causal impact, replication and institutional learning infrastructure.

## Quick start

```bash
python -m unittest discover -s tests -v
python src/realityloop_demo.py --scenario agent_assurance --out data/run.json
python src/realityloop_gateway.py
```

Open `prototype/index.html` for the offline demonstration.

## Evidence discipline

- I0 concept; I1 runnable; I2 controlled/synthetic; I3 single-site field inference; I4 independent replication; I5 multi-site transfer; I6 public-good standard.
- Evidence levels apply to claims, not whole repositories.
- H3/H4 harm overrides aggregate readiness and triggers KILL_AND_REPAIR.

## Safety

Research and governance prototype only. No autonomous clinical, financial, legal, military or rights-affecting action.
